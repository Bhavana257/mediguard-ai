# Contributing to MediGuard AI

Thank you for your interest in contributing to MediGuard AI!

## Development Setup

1. Fork the repository
2. Clone your fork
3. Set up backend and frontend as described in README.md
4. Create a feature branch: `git checkout -b feature/your-feature-name`

## Code Style

### Python
- Follow PEP 8 style guide
- Use type hints where possible
- Add docstrings to functions
- Keep functions focused and small

### TypeScript/React
- Use functional components with hooks
- Follow React best practices
- Use TypeScript for type safety
- Keep components small and focused

## Testing

Before submitting:
- Test with multiple patient IDs
- Verify all three agents work correctly
- Check frontend displays results properly
- Ensure error handling works

## Documentation

- Update README.md if adding features
- Update BACKEND.md or FRONTEND.md if changing functions
- Add comments for complex logic

## Pull Request Process

1. Ensure code follows style guidelines
2. Update documentation as needed
3. Test thoroughly
4. Submit PR with clear description
5. Address any review feedback
